Vehicle Trajectory Generator

  This vehicle trajectory generator utilizes a kinematic model of a vehicle to generate state and input trajectories of a vehicle. With an intuitive user interface, the simulation allows the user to effortlessly set starting and ending positions for a precise trajectory.

  This simulation excels in generating accurate trajectories based on user input, employing the physics-based bicycle model to provide a realistic representation of motion. The user-friendly design ensures a seamless experience throughout the process.
